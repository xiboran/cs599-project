# 研究生智能学习助手 - API接口规格说明书

## 1. 概述

本系统核心能力通过Python类接口提供，前端通过调用后端封装类实现交互。

## 2. 核心接口定义

### 2.1 对话主接口

- **类**：GradStudentAssistant
- **方法**：process_query(query: str) -> str
- **功能**：处理用户查询，整合记忆、Agent推理、工具调用后返回回答
- **输入参数**：
  - query: str，用户自然语言问题
- **返回值**：str，助手最终回答文本
- **错误处理**：内部捕获异常，返回友好错误提示

### 2.2 知识库检索接口

- **函数**：search_knowledge(query: str, top_k: int = 3) -> str
- **功能**：在向量库中检索相关知识片段
- **输入参数**：
  - query: str，检索关键词
  - top_k: int，返回结果数量，默认3
- **返回值**：str，格式化后的检索结果

### 2.3 用户画像接口

- **方法**：get_user_profile() -> str
- **功能**：获取当前用户的完整画像信息
- **返回值**：str，格式化的用户画像文本

### 2.4 记忆管理接口

- **方法**：reset_conversation() -> str
  - 功能：清空短期对话记忆
- **方法**：reset_profile() -> str
  - 功能：重置长期用户画像

## 3. 工具接口规范

所有工具函数统一为纯函数设计，输入输出均为字符串，便于Agent调用与解析。

## 4. 错误码说明

| 场景 | 处理方式 |
|------|----------|
| API密钥无效 | 返回鉴权失败提示 |
| 向量库不存在 | 返回知识库未初始化提示 |
| 工具调用异常 | 返回服务暂时不可用提示 |
