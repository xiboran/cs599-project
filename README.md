\# CS599 期末大作业：基于 Agentic AI 的研究生智能学习助手

课程：企业级应用软件设计与开发（AI 驱动的软件开发与 Agentic AI）

方向：方向一：Agentic AI 原生开发

学期：2025-2026 春季学期



\## 一、项目简介

本项目面向研究生科研与课程学习场景，基于 LangGraph 搭建多智能体协作系统，融合 RAG 检索增强、Function Calling 工具调用、会话记忆三大核心 Agent 技术，实现课程答疑、文献总结、个性化复习规划三类核心功能，完整落地 SDD 规格驱动开发工程流程。



\## 二、核心技术栈

\- 开发语言：Python 3.11

\- Agent 框架：LangChain + LangGraph

\- LLM 服务：DeepSeek API

\- 向量数据库：ChromaDB

\- 前端交互：Streamlit

\- 工程化工具：Git + Docker



\## 三、四大智能体功能

1\. 课程助手Agent：读取本地课件PDF，解答专业课考点、查询考试安排

2\. 论文助手Agent：解析文献内容、生成论文摘要、解释专业概念

3\. 学习规划Agent：结合用户记忆画像，生成阶段性复习计划表

4\. 路由调度Agent：自动识别用户问题，分发至对应功能子智能体
